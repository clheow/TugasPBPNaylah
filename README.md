Tugas PHP – Tugas 3, 4(database), dan 5
Tugas proyek berisi tiga tugas pemrograman PHP yang dijalankan menggunakan XAMPP dengan cara menyalin file ke folder htdocs.

Cara Menjalankan:

Memastikan XAMPP sudah terinstall dan Apache + MySQL aktif.
Menyalin file PHP ke folder C:/xampp/htdocs/tugaspbp.
Mengakses web melalui browser: http://localhost/tugaspbp/database/webview/index.php
